# 🏙 CityBuilder Bot (v1.0)
Интерактивный симулятор мэра в Telegram. Стройте улицы, возводите здания, собирайте налоги и следите за счастьем горожан!

## 🚀 Основные возможности
* **Динамическая экономика:** Доход зависит от налоговой ставки и счастья жителей.
* **Система уровней:** Разблокируйте новые здания (Заводы, Парки, Больницы) по мере роста.
* **Прокачка:** Улучшайте каждое здание до максимума, увеличивая его вместимость.
* **Случайные события:** Фестивали, визиты пришельцев или экономические кризисы.
* **Соревнования:** Глобальный топ богатейших мэров.

## 🛠 Технологии
* **Python 3.10+**
* **Aiogram 3.x** (Asynchronous Telegram Framework)
* **Aiosqlite** (Async Database)
* **Python-dotenv** (Управление секретами)
