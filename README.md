---

## 🚀 Основные изменения в новой версии

Проект прошел через полную архитектурную пересборку для обеспечения стабильности и масштабируемости:

### 💾 Система хранения и данных
* **Persistent Storage:** Полный переход на асинхронную базу данных **SQLite** (`aiosqlite`). Весь прогресс (деньги, уровни, здания, улицы) теперь сохраняется между перезагрузками бота.
* **Smart Middleware:** Внедрена `LoadCityMiddleware`, которая автоматически подгружает данные игрока из БД в каждое событие.
* **Авто-миграции:** Функция `update_db_structure()` автоматически обновляет таблицы при добавлении новых игровых параметров.

### 📈 Глубокая экономика и ООП
* **Классовая модель:** Логика зданий, улиц и города полностью инкапсулирована в классы внутри `classes_19.py`.
* **Налоговая политика:** Игрок может устанавливать ставку налога (от 1% до 25%), что напрямую влияет на индекс счастья жителей и доход.
* **Экспоненциальный рост:** Характеристики зданий (доход, жители) растут нелинейно в зависимости от уровня по формуле: `base * (level^1.1)`.

### 🎮 Игровые механики
* **Система уровней (XP):** За сбор налогов и строительство начисляется опыт. Повышение уровня открывает доступ к новым постройкам.
* **Случайные события:** Система ивентов в `events_15.py` (фестивали, инвесторы, катастрофы) создает динамичные условия для игры.
* **Глобальный Топ:** Реализована система лидербордов для сравнения капиталов разных мэров.

## 🛠 Стек технологий

* **Core:** Python 3.11+
* **Telegram API:** `aiogram 3.x`
* **Database:** `aiosqlite`
* **Environment:** `python-dotenv`

## 🏗 Структура проекта

| Файл | Описание |
| :--- | :--- |
| `city_bot.py` | Точка входа, обработка команд и FSM-состояний. |
| `classes.py` | Описание объектов `City`, `Street` и `Building`. |
| `DataBase.py` | Асинхронные методы взаимодействия с SQLite. |
| `config.py` | Баланс игры: цены, иконки и параметры зданий. |
| `middlewares.py` | Прослойка для инъекции данных города в хэндлеры. |
| `events.py` | Логика случайных игровых происшествий. |

## 📦 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/Nikson-007/telegram-city-builder.git](https://github.com/Nikson-007/telegram-city-builder.git)
   cd telegram-city-builder

2. **Установите необходимые зависимости:**
   ```bash
   pip install aiogram aiosqlite
   pip install -r requirements.txt

3. **Настройте переменные окружения:**
   Создайте файл `.env` и добавьте:
   ```env
   BOT_TOKEN=ваш_токен_от_BotFather
   ADMIN_ID=ваш_телеграм_id

4. **Запуск**
   ```bash
   python city_bot.py
